# ai-spend-audit
Credex AI-Audit is a web-based tool designed to help startups and teams audit their growing AI tech stacks. By analyzing subscriptions to services like ChatGPT, Claude, Cursor, and GitHub Copilot, the platform identifies cost-saving opportunities and provides smarter tool recommendations.

# AI Spend Optimization Audit Platform

The AI Spend Optimization Audit Platform is a website that helps companies look at how much they're spending on artificial intelligence software. It finds out if they are using tools that do the thing identifies licenses that are not being used enough and figures out how much money they can save. This platform is for companies, teams and organizations that have a lot of artificial intelligence subscriptions for their employees.

The platform has a tool that looks at the intelligence spending and finds out the risks detects waste and finds out if there are tools that are being used for the same thing. It also gives reports that can be downloaded and suggestions on how to make things better through a dashboard that is easy to use.

---

# Features

- The platform looks at how much money's being spent on artificial intelligence tools

- It keeps track of how many people are using each tool

- It estimates how much money is being wasted every month and every year

- It gives a score for how well the tools are being used and how much risk there is

- It finds out if there are tools that are being used for the same thing

- It gives suggestions on how to make things better

- It allows users to download reports as PDF and CSV files

- It can send reports by email

- It has a dashboard that is interactive and easy to use

- It has a modern user interface that works well on different devices
